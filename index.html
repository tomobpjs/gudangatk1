Koneksi Database Terblokir

Firebase menolak permintaan data karena Security Rules belum diizinkan. Ikuti 3 langkah cepat ini agar aplikasi bisa menyimpan ke Cloud:

1

Buka Firebase Console, pilih Project Anda, lalu klik menu Firestore Database.

2

Klik tab Rules (Aturan) dan ganti kodenya dengan ini:

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if true;
    }
  }
}
3

Klik tombol Publish di Firebase. Tunggu 30 detik, lalu Refresh halaman ini.
